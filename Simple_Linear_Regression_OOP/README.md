# Simple Linear Regression using OOP

## Project Overview
This project demonstrates Simple Linear Regression using Python and Scikit-Learn with an Object-Oriented Programming (OOP) approach.

## Features
- Data Exploration
- Train-Test Split
- Model Training
- Model Evaluation
- Prediction
- Regression Line Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Evaluation Metrics
- R² Score
- RMSE

## Learning Outcomes
- Understanding Simple Linear Regression
- Applying OOP concepts in Machine Learning
- Model Evaluation and Visualization
